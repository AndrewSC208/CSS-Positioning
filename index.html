CSS Positioning

Fixed Positioning 'fixes' the position of an element relative to the browser window. The element always stays fixed in place, even when scrolling.

This is how you can create elements that stay on the page even when scrolling, aka nav bar buttons, scrolling

position: fixed; top: 150px; left:px;

Absolute positioning: Takes an element out of document flow, meaning the browser acts as if the element has no width and height, and the other elements on the page move up as if it was never there. The position of the element is then fixed relative to the top level container, or the closest parent with a set positioning.

Note: The positioning is based off of the document, and not off of the original position unlike relative positioning.

Absolute positioning

Relative positioning: sets the position of an element relative to its original position. The element's original width and height is retained in document flow and other elements behave as if it was in its original position.

note: The big difference between realtive positioning nad absolute is that the original space is maintained with relative positioning. Position is also calculated baised off of its original position.

Static positioning: is the default behavior of elements. Inherited positioning tells and element to inhearit its prositioning form its parent element. Z-index controls the vertical stacking order of elements. Elements must have a set positioning for z-index to work.

Foating left: removes an element from normal document flow. It then takes that element and pushes it to the far left as possible. Other elements will move up as far as possiable to flow around the element and take its original spacd.

Floating right: removes an element from normal document floa. IT then pushes the element to the far right as far right. Other elements will morve up as far as possible to flow around the element and take its origianl space.

Floating: positions an element according to document flow, and then moves it to the far left or right as possible. elements following a floated element will move up as far as possible.

how to center and element correctly:

One of the most commonly asked questions is how to center elements since there is no option to float center. But by thinking carefully about how elements behave, we can use positioning to achieve he same result.

Center horizontally:

margin: 0 auto;

this will center anything. Note: need to have a set width for the element that you are centering

Center vertically and horizontally:

Note: If you just want to center vertically just add the first section to the container

Add this to the container:

position: absolute;
top: 50%;
mafrgin-top: (half of the size of the element you are centering vertically);
left: 0;
width: 100%;

Add this to element in container:

margin: 0 auto;
